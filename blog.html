Backpropagation in a Two-Layer Network

A complete derivation of weight and bias gradients with cross-entropy loss — from forward pass to parameter updates.

Setup & Notation

We'll derive every gradient for a two-layer fully-connected network trained with cross-entropy loss. This is the simplest architecture that demonstrates all the mechanics of backpropagation — once you understand it here, the extension to deeper networks is purely mechanical.

Our network has an input $\mathbf{x} \in \mathbb{R}^{n_0}$, one hidden layer with $n_1$ units, and an output layer with $C$ classes. The learnable parameters are:

$$W^{(1)} \in \mathbb{R}^{n_1 \times n_0}, \quad \mathbf{b}^{(1)} \in \mathbb{R}^{n_1}, \quad W^{(2)} \in \mathbb{R}^{C \times n_1}, \quad \mathbf{b}^{(2)} \in \mathbb{R}^{C}$$

We use $\sigma$ for the hidden-layer activation (sigmoid, ReLU, etc.) and softmax for the output layer. The goal is to derive $\frac{\partial \mathcal{L}}{\partial W^{(l)}}$ and $\frac{\partial \mathcal{L}}{\partial \mathbf{b}^{(l)}}$ for each layer $l \in \{1, 2\}$.

Two-Layer Architecture

The Forward Pass

The forward pass maps an input vector $\mathbf{x}$ to a probability distribution $\hat{\mathbf{y}}$ over $C$ classes in two stages.

Layer 1: Hidden Layer

We compute the pre-activation $\mathbf{z}^{(1)}$ and pass it through a nonlinearity $\sigma$ to get the hidden representation $\mathbf{a}^{(1)}$:

$$\mathbf{z}^{(1)} = W^{(1)} \mathbf{x} + \mathbf{b}^{(1)}$$
$$\mathbf{a}^{(1)} = \sigma\!\left(\mathbf{z}^{(1)}\right)$$

Each element $a_j^{(1)} = \sigma(z_j^{(1)})$ is computed independently. For sigmoid, $\sigma(z) = \frac{1}{1 + e^{-z}}$, with the convenient derivative $\sigma'(z) = \sigma(z)(1 - \sigma(z))$.

Layer 2: Output Layer

The hidden activations feed into the output layer, and softmax converts raw logits into probabilities:

$$\mathbf{z}^{(2)} = W^{(2)} \mathbf{a}^{(1)} + \mathbf{b}^{(2)}$$
$$\hat{y}_c = \text{softmax}(\mathbf{z}^{(2)})_c = \frac{e^{z_c^{(2)}}}{\displaystyle\sum_{j=1}^{C} e^{z_j^{(2)}}}$$

The output $\hat{\mathbf{y}} \in \mathbb{R}^C$ is a valid probability distribution: every entry is positive and they sum to one.

Forward Pass Data Flow

Cross-Entropy Loss

Given a one-hot ground-truth label $\mathbf{y}$ (where $y_c = 1$ for the correct class and $0$ elsewhere), the cross-entropy loss measures how far the predicted distribution is from the true one:

$$\mathcal{L} = -\sum_{c=1}^{C} y_c \log \hat{y}_c$$

Since $\mathbf{y}$ is one-hot, this simplifies to $\mathcal{L} = -\log \hat{y}_k$ where $k$ is the true class. The loss is zero when the model is perfectly confident in the correct class ($\hat{y}_k = 1$) and grows unbounded as $\hat{y}_k \to 0$.

A key result that simplifies all downstream gradients: the derivative of the cross-entropy loss composed with softmax has an elegantly simple form. If we define $\boldsymbol{\delta}^{(2)} = \frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(2)}}$, then:

$$\boldsymbol{\delta}^{(2)} = \hat{\mathbf{y}} - \mathbf{y}$$

This is the softmax-cross-entropy gradient — the error signal is simply the difference between predicted and true distributions. Everything that follows is applying the chain rule to push this signal backward through the network.

Training Loss Curve

Backpropagation: Output Layer

With $\boldsymbol{\delta}^{(2)} = \hat{\mathbf{y}} - \mathbf{y}$ in hand, we derive the gradients for $W^{(2)}$ and $\mathbf{b}^{(2)}$.

Gradient of $W^{(2)}$

Since $\mathbf{z}^{(2)} = W^{(2)} \mathbf{a}^{(1)} + \mathbf{b}^{(2)}$, the chain rule gives us:

$$\frac{\partial \mathcal{L}}{\partial W^{(2)}} = \boldsymbol{\delta}^{(2)} \, \mathbf{a}^{(1)\top} = \left(\hat{\mathbf{y}} - \mathbf{y}\right) \mathbf{a}^{(1)\top}$$

This is an outer product: $\frac{\partial \mathcal{L}}{\partial W^{(2)}} \in \mathbb{R}^{C \times n_1}$, matching the shape of $W^{(2)}$. Each entry $\frac{\partial \mathcal{L}}{\partial W_{ij}^{(2)}} = \delta_i^{(2)} \cdot a_j^{(1)}$ — the error at output $i$ times the activation at hidden unit $j$.

Gradient of $\mathbf{b}^{(2)}$

Since the bias enters $\mathbf{z}^{(2)}$ additively, its gradient is just the error signal itself:

$$\frac{\partial \mathcal{L}}{\partial \mathbf{b}^{(2)}} = \boldsymbol{\delta}^{(2)} = \hat{\mathbf{y}} - \mathbf{y}$$

Gradient Descent on the Loss Surface

Backpropagation: Hidden Layer

To update $W^{(1)}$ and $\mathbf{b}^{(1)}$, we need to propagate the error signal backward through the output weights and the activation function. This is where the "back" in backpropagation happens.

Error Signal at the Hidden Layer

First, we propagate $\boldsymbol{\delta}^{(2)}$ back through $W^{(2)}$ to get the error at the hidden activations:

$$\frac{\partial \mathcal{L}}{\partial \mathbf{a}^{(1)}} = W^{(2)\top} \boldsymbol{\delta}^{(2)} = W^{(2)\top} \left(\hat{\mathbf{y}} - \mathbf{y}\right)$$

Then we push through the activation function using the element-wise (Hadamard) product $\odot$:

$$\boldsymbol{\delta}^{(1)} = \frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(1)}} = \frac{\partial \mathcal{L}}{\partial \mathbf{a}^{(1)}} \odot \sigma'\!\left(\mathbf{z}^{(1)}\right) = W^{(2)\top}\!\left(\hat{\mathbf{y}} - \mathbf{y}\right) \odot \sigma'\!\left(\mathbf{z}^{(1)}\right)$$

The $\sigma'(\mathbf{z}^{(1)})$ term gates the gradient: for sigmoid, units near saturation ($z \to \pm\infty$) have $\sigma' \approx 0$, so they receive almost no update. For ReLU, $\sigma'(z) = \mathbb{1}[z > 0]$, so dead neurons (with $z \leq 0$) get zero gradient.

Gradient of $W^{(1)}$

Now the pattern is identical to the output layer — an outer product of the error signal with the input to this layer:

$$\frac{\partial \mathcal{L}}{\partial W^{(1)}} = \boldsymbol{\delta}^{(1)} \, \mathbf{x}^\top$$

This gives $\frac{\partial \mathcal{L}}{\partial W^{(1)}} \in \mathbb{R}^{n_1 \times n_0}$, matching the shape of $W^{(1)}$.

Gradient of $\mathbf{b}^{(1)}$

And the bias gradient is again just the error signal:

$$\frac{\partial \mathcal{L}}{\partial \mathbf{b}^{(1)}} = \boldsymbol{\delta}^{(1)}$$

Parameter Updates

With all four gradients computed, we update each parameter using gradient descent with learning rate $\eta$:

$$W^{(2)} \leftarrow W^{(2)} - \eta \frac{\partial \mathcal{L}}{\partial W^{(2)}} = W^{(2)} - \eta \left(\hat{\mathbf{y}} - \mathbf{y}\right) \mathbf{a}^{(1)\top}$$
$$\mathbf{b}^{(2)} \leftarrow \mathbf{b}^{(2)} - \eta \frac{\partial \mathcal{L}}{\partial \mathbf{b}^{(2)}} = \mathbf{b}^{(2)} - \eta \left(\hat{\mathbf{y}} - \mathbf{y}\right)$$
$$W^{(1)} \leftarrow W^{(1)} - \eta \frac{\partial \mathcal{L}}{\partial W^{(1)}} = W^{(1)} - \eta \, \boldsymbol{\delta}^{(1)} \, \mathbf{x}^\top$$
$$\mathbf{b}^{(1)} \leftarrow \mathbf{b}^{(1)} - \eta \frac{\partial \mathcal{L}}{\partial \mathbf{b}^{(1)}} = \mathbf{b}^{(1)} - \eta \, \boldsymbol{\delta}^{(1)}$$

where $\boldsymbol{\delta}^{(1)} = W^{(2)\top}\!(\hat{\mathbf{y}} - \mathbf{y}) \odot \sigma'(\mathbf{z}^{(1)})$. This is one complete step of gradient descent. In practice, NeuralFlow uses Adam with decoupled weight decay, but the gradient computation is identical — only the step size and momentum terms differ.
